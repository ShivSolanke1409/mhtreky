# 📌 Project Overview

## 🎯 Project Name

MHTREKY (Django-Based Web Application)



## 🧩 Description

This is a full-stack web application built using Django that allows users to explore, view, and book featured camps through an interactive and responsive interface.

The platform focuses on performance-driven UI, clean backend logic, and improved user experience through scroll-based parallax effects and dynamic notifications.

It demonstrates practical implementation of backend architecture, template rendering, dynamic UI interaction, and user feedback handling.



## 🛠 Tech Stack

### Backend

* Python
* Django
* Django Messages Framework

### Database

* SQLite (development)
* Easily configurable for PostgreSQL/MySQL

### Frontend

* HTML5
* Tailwind CSS
* JavaScript (Vanilla JS)
* Scroll-based parallax logic
* Auto-dismiss toast notification system

### Version Control

* Git
* GitHub



## 🚀 Core Features

### 1️⃣ Dynamic Hero Section

* Scroll-based Y-axis parallax effect
* Layered depth illusion (image + overlay text movement)
* Optimized transform-based animation (GPU accelerated)

### 2️⃣ Featured Camps Section

* Dynamic rendering using Django templates
* Responsive grid layout
* Clean UI structure

### 3️⃣ Booking Flow

* Form handling via Django views
* Server-side validation
* Feedback via Django messages framework

### 4️⃣ Smart Notification System

* Context-based styling (success, error, warning)
* Auto-dismiss with fade-out animation
* Hover pause functionality
* Clean DOM removal

### 5️⃣ Performance Optimization

* Transform-based animations (no layout thrashing)
* Overflow control for parallax safety
* Conditional rendering
* Clean separation of logic and UI
